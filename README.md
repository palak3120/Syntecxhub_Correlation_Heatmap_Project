# Correlation Heatmap & Pairwise Relationships

## Objective

To analyze relationships between numerical features of the Iris dataset using correlation analysis and visualization techniques.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Dataset

Iris Dataset

## Tasks Performed

1. Loaded the Iris dataset.
2. Computed Pearson correlation coefficients.
3. Created a correlation heatmap.
4. Generated pairwise relationship plots using Pairplot.
5. Analyzed positive and negative correlations among features.

## Results

* Petal length and petal width showed the strongest positive correlation (0.96).
* Sepal length was strongly correlated with petal length and petal width.
* Sepal width showed weak to moderate negative correlations with other features.
* Pairplot visualization confirmed these relationships.

## Conclusion

Correlation analysis helps identify relationships among variables. The Iris dataset shows strong associations among petal measurements, making them useful features for data analysis and machine learning tasks.
